# Ottonova Maps API

## Overview
This returns the city maps.

## Installation
* `git clone https://github.com/kinglarce/ottonova-maps-api.git`
* `cd ottonova-maps-api`
* `npm install`

### Running Locally
* Dev: `npm run dev`
* Tests: `npm run test`
* URL: `http://localhost:8000/`

### Running Publicly
* Deploy: `npm run build && npm run start`
